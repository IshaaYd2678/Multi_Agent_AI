Multi-Agent AI Document Processor

An intelligent multi-agent system that classifies, extracts, and processes documents including emails, JSON data, and PDFs.

📧

Email Agent

Extracts sender, urgency, and key details from email content.

🛠

JSON Agent

Parses and validates structured JSON data like invoices.

📄

PDF Agent

Reads and extracts text content from PDF documents.

Getting Started

  1. Clone the repository
  2. Install dependencies: pip install -r requirements.txt
  3. Run the Streamlit app: streamlit run streamlit_app.py
  4. Or run the CLI: python main.py
